Step 2

Download all four files and put them in the folder you chose above. Keep them there permanently — the LaunchAgent points to this location.

Don't move the files after installing. The LaunchAgent plist uses the absolute path you set. If you move the folder, you'll need to re-run the installer.

Step 3

    
Run in Terminal

    
Open Terminal (⌘ Space → "Terminal") and run these commands in order.


    

      ℹ️
      
Commands below use the install directory and API key you entered above. If you haven't filled them in yet, go back to Step 1.

    


    

      
    


  








  

    
Step 4

    
Grant macOS permissions

    
The tracker reads your browser tab using AppleScript. macOS blocks this by default.


    

      

        
A

        

          
Allow Terminal to control Safari

          
The first time the tracker runs, a system dialog will appear. Click OK.

          If no dialog appears: System Settings → Privacy & Security → Automation → Terminal → Safari ✓

        

      

      

        
B

        

          
Enable Safari developer features (for project name detection)

          
Without this, all your EasyEDA time shows under one project name instead of per-board.

          Safari → Settings (⌘,) → Advanced → ✓ Show features for web developers
